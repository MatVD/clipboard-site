Toutes les commandes (Android et IOS)

  • Double touch sur un mot puis glisser jusqu’a la fin de la phrase : la sélection est automatiquement coller dans le presse papier
  • Copier avec tag: Triple touch sur un mot puis glisser jusqu’a la fin de la sélection. Une petite fenêtre s’ouvre pour nommé le tag
  • Coller dans un dossier précis: Double touch sur le premier mot puis glisser jusqu’à la fin de la sélection et laisser le doigt appuyé longtemp. Le dossier du presse papier s’ouvre pour que vous puissiez choisir le dossier souhaité.
  • Vous pouvez également copier-coler des images

Toutes les commandes et raccourcis clavier

Ensembles des raccourcis sous Windows :

Ctrl + C pour copier des données dans le presse-papier

Ctrl + X pour couper des données vers le presse-papier

Ctrl + V pour coller des données depuis le presse-papier

Ctrl + t pour copier avec un Tag (c’est-a-dire une balise poue le retrouver facilement)

Ctrl + d pour copier dans un dossier souhaité (Une fenêtre s’affiche avec les différents dossiers du presse papier)

Raccourcis alternatifs (disponibles dans de nombreux programmes sous Windows) :

Ctrl + Inser pour copier

Shift + Suppr pour couper

Shift + Inser pour coller

Les raccourcis standards de MacOS sont :

Command + C pour copier des données dans le presse-papier

Command + X pour couper des données vers le presse-papier

Command + V pour coller des données depuis le presse-papier

Command + t pour copier avec un Tag (c’est-a-dire une balise poue le retrouver facilement)

Command + d pour copier dans un dossier souhaité (Une fenêtre s’affiche avec les différents dossiers du presse papier)

Ctrl-k pour copier à partir du curseur jusqu'à la fin de ligne.

Ctrl-y pour couper du "kill-ring" au curseur.
